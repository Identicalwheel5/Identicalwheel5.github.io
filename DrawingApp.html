In our Android drawing app project, a key feature was the ability to save and upload drawing files. To achieve this, we implemented Google Firebase for cloud storage. Firebase provides a "Storage" service, which essentially acts as a file system in the cloud. We utilized this Firebase Storage to handle the uploading of the drawing files, using references to define the storage paths and manage the data. Furthermore, we incorporated Firebase's access control features, defining rules to manage how users could access these stored files.
